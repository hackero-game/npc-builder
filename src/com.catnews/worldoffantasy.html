**UPDATE** World of Fantasy update released

Snowstorm Entertainment has released a update to their classic World of Fantasy Game. They worked really hard to release this new version of the game a spokesman said. The developers worked day and night to make the release of the update as early as possible. To install the update start the launcher and pay $19,99 excl. taxes. The launcher will then automatically install the update.

Noteable changes

  • Free features are now paid
  • Online gameplay has been removed
  • Questgiver are now all replaced by pink unicorns
  • Fantasy setting was replaced by a science fiction setting

Snowstorm originally announced the update 3 Weeks ago on their FantasyCon. Fans were not amused by the changes however they expect to have all players buy the update otherwise the game will not work anymore.

If you would like to read more check out the official world of fantasy changenotes at snowstorm.com
